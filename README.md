## What is Zzik?

Zzik is a multilingual news aggregation service that:
- Collects news from RSS feeds worldwide
- Translates headlines into 7 languages (English, Korean, Chinese, Japanese, Russian, Thai, Vietnamese)
- Clusters articles into events and headlines
- Displays them on an interactive globe
- Provides real-time chat rooms
